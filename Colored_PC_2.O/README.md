# Script for .xyz format point cloud.

### Folder:

-   **[Current folder (Colored_PC_2.O)](Colored_PC_2.O)**: Contains all the Python scripts for .xyz format point cloud processing.
-   **[Data/data.xyz(Colored_PC_2.O)/Point_cloud(.xyz)](Data/data.xyz(Colored_PC_2.O)/Point_cloud(.xyz))**: This directory contains subfolders for each type of model (in .xyz format).
-   **[Data/data.xyz(Colored_PC_2.O)/Output_PC_Vis](Data/data.xyz(Colored_PC_2.O)/Output_PC_Vis)**: Output of this script, including rendered images and Blender files.

### Script Usage Example (e.g., for "cone"):

To run the script, you'll need to provide the following parameters:

1. `directory_path`: e.g. path to folder in which all the point cloud of "cone" is saved.

2. `ground_truth_filename`: name of ground truth file present in the `directory_path` e.g. "cone.xyz"

3. `output_dir`: path to folder where rendered image and blender file will be saved.
