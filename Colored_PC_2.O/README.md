# Script for .xyz formate point cloud.

### Folder:

`current folder` contain all the the python scripts.

[Data\ data.xyz(Colored_PC_2.O)\ Point_cloud(.xyz)](Data\data.xyz(Colored_PC_2.O)\Point_cloud(.xyz)) it contain subfolder for each type of model(.xyz formate).

`Output_PC_Vis` Output of Script.

### Lets us assume we want to run script for "cone".


1. `directory_path`: e.g. path to folder in which all the point cloud of "cone" is saved.

2. `ground_truth_filename`: name of ground truth file present in the `directory_path` e.g. "cone.xyz"

3. `output_dir`: path to folder where rendered image and blender file will be saved.
