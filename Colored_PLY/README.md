# Script for .ply format point cloud.

### Folder:

-   **[colored_ply](Colored_PLY)**: Contains all the Python scripts for .ply format point cloud processing.
-   **[Data\data.ply(Colored_PLY)\kinect\kinect](Data\data.ply(Colored_PLY)\kinect\kinect)**: This directory contains each type of model (in .ply format).
-   **[Data\data.ply(Colored_PLY)\Output](Data\data.ply(Colored_PLY)\Output)**: Output of this script, including rendered images and Blender files.

### Script Usage Example (e.g., for "sphere"):

To run the script, you'll need to provide the following parameters:

1. `project_root_cwd`: path to current working directory "Pyblender"

2. `input_folder`: path to `kinect` folder wherer all the .ply saved.

3. `output_folder`: path to folder where rendered image and blender file will be saved in their respective subfolder.
